# Como levantar el proyecto

## backend part
1. Descargar el proyecto
2. correr "npm install" para instalar paquetes
3. levantar proyecto con: 
```
npm start
```


## frontend part
1. Descargar el proyecto
2. correr "npm install" para instalar paquetes
3. levantar proyecto con: 
```
npm start
```

Para poder usar graphQL:  
```
http://localhost:4000/graphql
```
